# External Resources

## NCEI Water Column Sonar Viewer

The NCEI Water Column Sonar Viewer provides public access to over 300 terabytes of global sonar data used to map marine life, habitats, and undersea oil or gas leaks. Established in 2014 through a collaboration between NOAA and the University of Colorado, this archive centralizes acoustic data from government, academic, and international sources to facilitate broad scientific research. Users can explore these datasets via an interactive map with specialized filters for cruises, platforms, and instruments. It can be accessed here: [NCEI Water Column Sonar Viewer](https://www.ncei.noaa.gov/maps/water-column-sonar/).
